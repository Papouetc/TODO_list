## Features

- **Adding and removing** - easily add remove note by a simple click
- **Marking done** - Mark done your notes when you dont need them anymore 
  
All of your notes will be automatically saved in LocalStorage so you don't need an account 
## Usage
Simply copy this repository in your local environment and open the index.html file.
