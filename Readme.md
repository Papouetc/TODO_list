## Features

- **Add and remove tasks** – Easily create or delete tasks with a single click.
- **Mark tasks as completed** – Keep track of what you've finished.
- **Automatic saving** – Tasks are stored in LocalStorage, no account required.

## Usage

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser
